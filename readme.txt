Proyecto API petsore 06/08/2023

Proyecto de pruebas automatizado API sobre flujo de Agregar, consultar y actualizar los datos de las mascotas en la petstore.
El proyecto petstore consta de la ejecución de 4 casos de pruebas, que cumple con la funcionalidad de añadir una nueva mascota a la tienda, realizar consultas por id y status y actualizar nombre y estatus de las mascotas existentes

El proyecto se desarrolla bajo java versión "17.0.1"

Tener en cuenta para la ejecución del proyecto:
- Importar este como proyecto karate
- Ejecución se realiza por medio de los features en casa test ubicados en la ruta ../src/test/java/Pet
- Se debe tener en cuenta el orden de ejecución especificado y observaciones mencionadas en cada test
- La configuración de la ejecución se realiza con los comandos de RUN sobre cada uno de los features

Para el informe de pruebas se puede abrir por consola una vez ejecutados los casos abriendo el archivo .html al final de la ejecución por consola.
también se puede abrir el reporte navegando en el proyecto buscando la ruta ..../target/karate-reports/karate-summary.html
